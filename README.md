# 🍽️ Kuliner Palembang

Website rekomendasi restoran khas Palembang berbasis **CodeIgniter 4**.  
Aplikasi ini memungkinkan pengguna mencari restoran, melihat detail, memberikan komentar, serta rating.

---

## ✨ Fitur Utama

- 🔐 Login & Register User
- 🏪 CRUD Data Restoran (Admin)
- ⭐ Sistem Rating (1–5 bintang)
- 💬 Komentar / Review Pengguna
- 🔍 Pencarian Restoran
- 📍 Google Maps Lokasi Restoran
- 📊 Dashboard Admin (statistik)
- 🖼️ Upload Gambar Restoran

---

## 🖼️ Tampilan Aplikasi

### 🏠 Homepage
Menampilkan daftar restoran berdasarkan rating tertinggi

### 📄 Detail Restoran
- Informasi lengkap
- Rating & jumlah review
- Komentar pengguna
- Peta lokasi (Google Maps)

### 🔑 Login & Register
Sistem autentikasi sederhana

### 🛠️ Admin Dashboard
- Kelola restoran
- Lihat statistik data
- Monitoring komentar

---

## 🧰 Teknologi yang Digunakan

- ⚙️ CodeIgniter 4
- 🐘 PHP 8+
- 🗄️ MySQL
- 🎨 Bootstrap / AdminLTE
- 🧠 JavaScript
- 🗺️ Google Maps Embed API
